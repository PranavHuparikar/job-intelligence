6. What is cross-encoder reranking, and why does it improve RAG performance over bi-encoder retrieval alone? — Tests: Retrieval depth; bi-encoder = fast recall, cross-encoder = precise precision on top-K

7. Walk me through how HyDE (Hypothetical Document Embeddings) improves retrieval. — Tests: Advanced RAG techniques; LLM generates hypothetical answer, embed that, retrieve similar real docs

8. How do you evaluate a production RAG system? What does RAGAS measure? — Tests: Evaluation maturity; answer: faithfulness, answer relevance, context precision, context recall

9. How would you design a multi-agent system that prevents an agent from getting stuck in an infinite loop? — Tests: Production robustness thinking; answer with retry limits, fallback edges, timeout states in LangGraph

10. If a supervisor agent needs to decide at runtime between parallel execution and sequential execution, how do you implement that in LangGraph? — Tests: Advanced LangGraph; answer with conditional edges returning either a list (parallel Send) or a string (sequential routing)
