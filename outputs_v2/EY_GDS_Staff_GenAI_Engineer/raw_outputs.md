What are the tradeoffs between fixed-size, semantic, and sentence-window chunking? — Tests: RAG fundamentals; fixed-size = simple, loses context at boundaries; semantic = better coherence, more compute; sentence-window = good for conversational retrieval

3. Walk me through a complete prompt engineering iteration cycle. How do you know when a prompt is good enough? — Tests: Prompt iteration discipline; answer: define eval criteria first, test on representative samples, measure faithfulness/relevance, iterate on system prompt and few-shots until threshold met

4. What is Chain-of-Thought prompting and when would you use it vs. direct prompting? — Tests: Prompt techniques; CoT = ask model to reason step-by-step before answering; use for multi-step reasoning, math, complex decisions; not needed for simple classification

5. How do you implement guardrails in a GenAI pipeline? What specifically do you check on input vs. output? — Tests: Safety implementation; input: PII detection, injection attacks, off-topic queries; output: faithfulness check, PII in response, toxic content filter

8. How do you add logging and monitoring to a GenAI pipeline? What signals matter? — Tests: LLMOps good-to-have; answer: log per-request latency, token usage, faithfulness score, retrieval quality, error rate; your L1-L4 quality indicator system is a direct answer
